## About

UniClub Dynamic is a comprehensive club management system for the University of Mindanao, featuring:

- **Student Module**: Discover clubs, join clubs, RSVP to events, earn points, and communicate with officers
- **Officer Module**: Manage club members, events, attendance, finances, and communications
- **Admin Module**: System-wide management, approvals, analytics, and user management

## Features

- Club discovery and membership management
- Event management with RSVP system
- Student points and rewards system
- Messaging system between students, officers, and admins
- Attendance tracking
- Financial management for clubs
- Role-based access control
- Secure authentication and authorization
